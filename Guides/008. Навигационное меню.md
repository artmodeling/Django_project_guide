# Навигационное меню

1. Внести изменения в файл «store/urls.py»:  
\- в строку `path('products/add/', views.create_product),` добавить запятую, пробел и `name="new_product"`  
\- в результате строка должна выглядеть так:  
`path('products/add/', views.create_product, name="new_product"),`  

2. Внести изменения в файл «store/templates/base.html»:  
	1) удалить строку `<header>Общая шапка для сайта</header>`,  
	2) вместо строки, которую удалили, записать код:
	``` 
   <header>
      <nav>
        <a href="{% url "product_list" %}">Все товары</a>
        <a href="{% url "new_product" %}">Добавить новый товар</a>
      </nav>
    </header>
	``` 

3. Проверить на сайте, что внесенные изменения работают.

4. (Опционально.) Создать коммит: «feat(nav): Добавил навигационное меню.».
