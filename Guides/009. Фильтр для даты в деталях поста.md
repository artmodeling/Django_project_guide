# Фильтр для даты в деталях поста

1. Внести изменения в файл «store/templates/store/product_detail.html»:  
\- В фигурных скобках в строке «`<p>{{ product.created_at }}</p>`» дописать: «`|date:"d.m.Y H:i"`»  
\- В результате строка должна выглядеть так:  
``` 
<p>{{ product.created_at|date:"d.m.Y H:i" }}</p>
``` 

2. (Опционально.) Создать коммит: «feat(dj-filter): Добавил фильтр для даты в деталях товара.»
