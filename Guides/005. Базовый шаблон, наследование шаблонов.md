# Базовый шаблон, наследование шаблонов

1. Создать шаблон для шаблонов файлов .html. Этот файл создается в самой папке  «templates», а не в папке приложения. На основе этого шаблона будут создаваться новые шаблоны .html:  
    1) Создать файл «store/templates/base.html».  
    2) В этой файл «store/templates/base.html» записать код:  
    ```html
    <!DOCTYPE html>
    <html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>{% block page_title %}{% endblock page_title %}</title>
    </head>
    <body>
        <header>Общая шапка для сайта</header>
        <main>{% block page_content %}{% endblock page_content %}</main>
        <footer>&copy; Все права защищены 2025</footer>
    </body>
    </html>
    ```  

**Примечание.** Далее нужно будет отредактировать уже созданные шаблоны html для того, чтобы они брали основные элементы для своей страницы html из шаблона для html.

2. Внести изменения в файл «store/templates/store/product_detail.html»:
	1) Удалить строки, которые сейчас есть в этом файле, то есть, эти:   
    ```html
    <!DOCTYPE html>
    <html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>{{ product.product_name }}</title>
    </head>
    <body>
        <a href="{% url 'product_list' %}">&larr;Назад</a>

        <h1>Детали товара {{ product.product_name }}:</h1>
        <h2>ID товара: {{ product.id }}</h2>
        <h3>{{ product.product_name }}</h3>
        <p>{{ product.about_item }}</p>
        <p>{{ product.created_at }}</p>  
    </body>
    </html>
    ```
    2) Записать в этот файл код:  
    ```html
    {% extends "base.html" %}

    {% block page_title %}{{ product.product_name }}{% endblock page_title %}

    {% block page_content %}
    <a href="{% url 'product_list' %}">&larr;Назад</a>

    <h1>Детали товара :</h1>
    <h2>ID товара: {{ product.id }}</h2>
    <h3>{{ product.product_name }}</h3>
    <p>{{ product.about_item }}</p>
    <p>{{ product.created_at }}</p>
    {% endblock page_content %}

    ```   

3. Внести изменения в файл «store/templates/store/product_list.html»:
	1) Удалить все строки, которые есть в этом файле, то есть, вот эти:  
    ```html 
    <!DOCTYPE html>
    <html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Все товары</title>
    </head>
    <body>
        <h1>Все товары</h1>
        {% comment %}
            Пример
            многострочного
            комментария.
            Всем привет!    
        {% endcomment %}
        {# Пример однострочного комментария. #}
        {% if products %}
            {% for product in products  %}
                <div class="product-container" style="border: 1px solid black; margin-bottom: 10px; padding: 15px;">
                    {% comment %} <h2>Наименование товара: <a href="/products/{{ product.id }}/">{{ product.product_name }}</a></h2> {% endcomment %}
                    <h2>Наименование товара: <a href="{% url 'product_detail' product_id=product.id %}">{{ product.product_name }}</a></h2>
                    <p>Подробнее о товаре: {{ product.about_item }} </p>
                    <small>Дата добавления товара на сайт: {{ product.created_at }} </small>
                </div>
            {% endfor %}
        {% else %}
            <p>Страница находится в разработке. Пожалуйста, зайдите к нам позже.</p>
        {% endif %}

        
    </body>
    </html>

    ``` 
	2) Записать в этот файл код:  
    ```html 
    {% extends "base.html" %}

    {% block page_title %}Все товары{% endblock page_title %}

    {% block page_content %}
    <h1>Все товары</h1>

    {# Отображение товаров #}
    {% if products %}
        {% for product in products %}
        <div class="product-container" style="border: 1px solid black; margin-bottom: 10px; padding: 15px;">
            {% comment %} <h2>Наименование товара: <a href="/products/{{ product.id }}/">{{ product.product_name }}</a></h2> {% endcomment %}
            <h2>Наименование товара:  <a href="{% url 'product_detail' product_id=product.id %}">{{ product.product_name }}</a></h2>
            <p>Подробнее о товаре:  {{ product.about_item }}</p>
            <small>Дата добавления товара на сайт: {{ product.created_at }}</small>
        </div>
        {% endfor %}
    {% else %}
        <p>Страница находится в разработке. Пожалуйста, зайдите к нам позже.</p>
    {% endif %}
    {% endblock page_content %}

    ``` 

4. Внести изменения в файл «config/settings.py»:  
	\- Найти список «TEMPLATES».  
    \- В списке «TEMPLATES» найти строку `'DIRS': [],` в этой строке внутри квадратных скобок записать, скопировав все кавычки и пробелы:  
`BASE_DIR / 'templates/'`  
    \- В результате эта строка должна выглядеть следующим образом:  
    `'DIRS': [BASE_DIR / 'templates/'],`

5. Проверить на сайте, что страница html, для которой вносили изменения, работает.

6. (Опционально.) Создать коммит:  
«refactor(store): Наследование шаблонов:  
\- добавил базовый шаблон  
\- наследовал от базового шаблона остальные шаблоны»
