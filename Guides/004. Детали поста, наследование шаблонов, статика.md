3. Редактирование файла «store/views.py»:  
	1) в конце строки `from django.shortcuts import render` добавить запятую с пробелом и дописать `get_object_or_404`  
	2) вместо строки, которую удалили, вставить строку –  
	`from django.shortcuts import render, get_object_or_404`  
	3) в самом конце файла добавить две пустые строки для того, чтобы визуального разделения функций, и после этих пустых строк вставить код, соблюдая отступы слева –
```python 
def get_product_detail(request, product_id):
  # return render(request, 'store/product_detail.html', {"product": Products.objects.get(id=product_id)})
  return render(request, 'store/product_detail.html', {"product": get_object_or_404(Products, id=product_id)})
```
