# Подключение статики, core.css, подключение к base.html

1. Создать файл «файл «static/css/core.css»:
    1) В самом проекте «Online_store» создать папку «static», в этой папке создать еще одну папку «css». 
    2) В папке «css» создать файл «static/css/core.css» и записать в него код:  
    ```css 
    * {
    padding: 0;
    margin: 0;
    box-sizing: border-box;
    }

    ``` 

2. Внести изменения в файл «config/settings.py»:
	\- после строки «`STATIC_URL = 'static/'`» добавить строку «`STATICFILES_DIRS = [BASE_DIR / 'static/']`»

3. Внести изменения в файл «store/templates/base.html»:  
	1) В самом начале файла перед строкой «`<!DOCTYPE html>`» добавить строку вместе с пустой строкой после нее:  
    ```html
    {% load static %}

    ``` 

    После этих изменений код будет выглядеть следующим образом:  

    ```html
    {% load static %}

    <!DOCTYPE html>
    <html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>{% block page_title %}{% endblock page_title %}</title>
        <link rel="stylesheet" href="{% static 'css/core.css' %}">
    </head>
    <body>
        <header>Общая шапка для сайта</header>
        <main>{% block page_content %}{% endblock page_content %}</main>
        <footer>&copy; Все права защищены 2025</footer>
    </body>
    </html>
    ```

	2) После строки «`<title>{% block page_title %}{% endblock page_title %}</title>`» добавить строку:  
    ```html 
    <link rel="stylesheet" href="{% static 'css/core.css' %}">
    ``` 
4. (Опционально.) Создать коммит:   
«feat(static):  
\- Подключил статику  
\- Создал core.css и подключил к base.html»
